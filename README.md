# Portfolio (single-page) — Next.js + Tailwind

This is a lightweight one-page portfolio in the same *structure* as the reference link: hero → about → work → certifications → skills → projects → footer.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Customize content

Edit:

- `data/site.ts` (name, about, experience, skills, projects, links)
- `public/projects/*` (replace placeholder SVGs with your screenshots)

## Deploy

Push to GitHub → import into Vercel → deploy.
